## Getting started

### Prerequisites

- Node

```
curl https://get.volta.sh | bash
volta install node
```

### Install dependencies

```
npm install
npm i -g firebase-tools
```

---

### Start the development server

```
npm start
```

### Build for production

```
npm run build
```

### Build for production and start preview server

```
npm run preview
```
